# Logo Setup Instructions

To add the GroupNest logo to your app, follow these steps:

## Step 1: Save the Logo Image

1. Save the GroupNest logo as a PNG file
2. Name it `logo.png`
3. Place it in the `assets/images/` folder

The logo should be:
- **Square format** (e.g., 1024x1024 pixels for best quality)
- **PNG format** with transparency support
- **High resolution** for crisp display on all devices

## Step 2: Verify File Location

Make sure the logo file is at:
```
assets/images/logo.png
```

## Step 3: App Icon Sizes (Optional - for better quality)

For the best app icon experience, you may want to create different sizes:

- **icon.png** - 1024x1024 (main app icon)
- **splash-icon.png** - 200x200 or larger (splash screen)
- **android-icon-foreground.png** - 1024x1024 (Android adaptive icon foreground)
- **android-icon-background.png** - 1024x1024 (Android adaptive icon background - solid color)
- **android-icon-monochrome.png** - 1024x1024 (Android monochrome icon)
- **favicon.png** - 32x32 or 48x48 (web favicon)

You can use the same logo.png for all of these, or create optimized versions.

## What's Already Configured

The app is already set up to use the logo in:
- ✅ Login screen
- ✅ Signup screen
- ✅ App icon (app.json)
- ✅ Splash screen (app.json)
- ✅ Android adaptive icons (app.json)
- ✅ Web favicon (app.json)

Once you add `logo.png` to `assets/images/`, everything will work automatically!
